# hitta-genus

Python CLI utility to look up Swedish noun genders on [svenska.se](https://svenska.se)

## Installation

```
# pip install git+git://github.com/Retzudo/hitta-genus#egg=hitta-genus
```

## Usage

```
$ genus hus
hus: ett hus, huset
$ genus mus
mus: en mus, musen
$ genus hus mus
hus: ett hus, huset
mus: en mus, musen
```
